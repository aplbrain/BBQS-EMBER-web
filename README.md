# BBQS-EMBER-web


## Prerequisities

- Node.js
- Yarn 

## Install


### Frontend

1. Install dependencies

    ```bash
    yarn install
    ```

## Development

### Frontend

1. Start the application in development mode

    ```bash
    yarn quasar dev
    ```

1. The frontend should now be running at [http://localhost:9000](). The application will reload as you make edits.
